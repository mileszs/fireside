fireside
========

A Reddit clone written with Camping

## Getting Started with this Bad Mofo

  git clone git@github.com:mileszs/fireside.git
  cd fireside
  bundle install
  bundle exec camping fireside.rb

Then visit `localhost:3301` in your browser.
